# سالن زیبایی پروانه اکبرپور

سیستم جامع مدیریت سالن زیبایی — اپلیکیشن اندروید با Kotlin و Jetpack Compose، بکند NestJS، و پنل مدیریت React.

## ساختار پروژه

```
├── backend/          # بکند NestJS + PostgreSQL
├── android/          # اپ اندروید Kotlin/Jetpack Compose  
├── admin/            # پنل مدیریت React
└── .github/workflows/ # CI/CD برای ساخت APK/AAB
```

به بخش [Releases](../../releases) بروید و آخرین APK را دانلود کنید.
